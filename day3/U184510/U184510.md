# 冒泡排序趟数期望

## 题目背景

Ran很喜欢【冒泡排序】，所以出了很多相关的题目，这又是其中一道。

对于一个排列 $a[1...n]$，进行一趟冒泡排序的代码为：
```cpp
for(int i=1;i<n;++i){
    if(a[i]>a[i+1]) swap(a[i], a[i+1]);
}
```
若排列在 $k$ 趟冒泡排序之后变为有序，则最小的 $k$ 定义为 $res$。

## 题目描述

给一个长度为 $n$ 的随机排列（ $n!$ 种情况等概率出现），请你计算 $res$ 的期望值，对 `1e9+7` 取模。

例如 $n=3$:
- $a = [1,2,3], res = 0$
- $a = [1,3,2], res = 1$
- $a = [2,1,3], res = 1$
- $a = [2,3,1], res = 2$
- $a = [3,1,2], res = 1$
- $a = [3,2,1], res = 2$

期望为 $(0+1+1+2+1+2)/6 = 7/6$

## 输入格式

输入一行1个整数 $n$

## 输出格式

输出1个整数代表答案
